# Student Management System

A full-stack Student Management System built using Spring Boot, Spring Data JPA, MySQL, HTML, CSS, and JavaScript. This application provides complete CRUD operations for managing student records through a clean and responsive user interface.

---

## Features

- Add Student
- View All Students
- Update Student Details
- Delete Student
- Responsive User Interface
- REST API using Spring Boot
- MySQL Database Integration
- Spring Data JPA
- MVC Architecture

## How to Run

1. Clone the repository

```bash
git clone https://github.com/K-SindhuRani/Student-Management-System-CRUD.git
```

2. Open the project in VS Code or IntelliJ IDEA.

3. Configure your MySQL credentials in `application.properties`.

4. Run the project using:

```bash
mvn spring-boot:run
```

5. Open your browser and visit:

```
http://localhost:8081
```
